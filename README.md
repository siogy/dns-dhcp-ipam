# DDI Solution

A standalone solution for DNS, DHCP, and IP Address Management using BIND9, ISC Kea-DHCP, TeemIP, and PacketFence.

## Components

- **DNS**: BIND9
- **DHCP**: ISC Kea-DHCP
- **IPAM**: TeemIP
- **Policy**: PacketFence
